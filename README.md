Implemented an Encoder-Decoder Transformer for English to Spanish translation from scratch in PyTorch. Uses multi-head cross-attention to align source and target sequences. Trained on 50,000 sentence pairs from the Anki dataset.


# Neural Machine Translation with Cross-Attention

English to Spanish translation using a Transformer Encoder-Decoder model.

## Overview
- Dataset: Anki Spanish-English (50,000 pairs)
- Architecture: Multi-Head Cross-Attention Transformer
- Framework: PyTorch
- d_model: 256 | Heads: 4 | Epochs: 20
